# Antigravity 2.0 汉化与体验优化补丁

这个项目是针对 Google 的智能体编程客户端 **Antigravity 2.0** 制作的无损汉化与体验优化补丁。

除了实现完整的界面中文显示外，还顺手解决了两个在使用客户端时最让人烦恼的痛点：**每次启动被迫开启代理的 TUN 模式** 以及 **从系统托盘恢复窗口时极慢且伴随 `loading` 刷新的问题**。

---

## 🚀 核心改进

### 1. 深度中文汉化（支持云端热更新）
* **不影响代码**：内置高精度扫描引擎，自动跳过 Monaco Editor 代码区、富文本输入框、日志输出面板，**绝对不会误翻译任何代码字符**，保障代码语法高亮与内容原汁原味。
* **热更新词库**：启动时使用本地缓存瞬间完成界面翻译，并在后台异步请求社区最新词库进行静默合并，后续无需重复安装即可获取最新校对。
* **全覆盖**：系统菜单栏、托盘指示器、设置项、新手引导弹窗以及 OS 标题栏全部中文化。

### 2. 免 TUN 模式代理穿透（✨ 独家痛点解决）
* 客户端启动时会自动探测 Chromium 的系统代理设置。
* 将解析出的代理地址自动以 `HTTP_PROXY` / `HTTPS_PROXY` 形式注入给后台智能体（`language_server`）进程。
* **您只需要像平常一样开启代理即可正常对话，不再需要特意去代理客户端里开启繁琐且耗资源的 TUN 模式！**

### 3. 0ms 瞬间还原（告别 Loading 重新加载）
* **隐藏代替销毁**：点击“关闭”按钮时，如果开启了“常驻后台”，窗口会被隐藏而不是销毁。
* **托盘秒开**：为系统托盘图标绑定了左键单击/双击事件。点击托盘图标时主窗口会在 0ms 内瞬间展示，**完全没有官方版烦人的 `loading antigravity` 重新加载进度条**，同时完美保留您关闭前没写完的输入框草稿。
* **彻底退出**：如果在设置中关闭了“常驻后台”，点击关闭会干净地退出程序，不残留任何后台进程。

### 4. 智能额度无感静默更新（✨ 独家首创）
* **打破官方限制**：彻底解决重启客户端额度卡死在 100% 或必须手动点开模型列表才刷新的痛点。
* **活跃即时刷新**：当您打字和对话期间（最近 15 秒有操作），后台会自动将刷新频率提升至 **每 5 秒一次**，额度扣减几秒内即可无感体现在左下角侧边栏小组件上。
* **闲置自动降频**：当您离开电脑时，后台自动切换至 **每 30 秒一次的安全低频** 轮询，完全规避因高频请求导致被谷歌云端接口临时限流封禁（429 Too Many Requests）的潜在安全风险。
* **高精 gRPC 解析**：主进程使用专用二进制 `application/grpc-web+proto` 封包格式通信，通过底层 `0x25` 字节偏移算法反序列化，数值与设置页面 100% 对齐。
* **首屏防闪烁**：启动加载时优雅显示 `--` 等待服务就绪，首屏 1 秒内无缝跳变至真实数值，去除了假 100% 闪烁。

### 5. 多账号极速切换与自动嗅探 (v1.1.2 新增)
* **自动嗅探持久化**：当您在官方客户端中点击“添加账号”并登录成功后，主进程会在后台自动嗅探最新的登录状态与凭据，截断无效冗余字符后持久化保存至本地独立文件夹中。
* **左下角秒级切换**：在客户端左下角（设置上方）直接集成账号下拉菜单，点击任意账号即可自动覆盖系统 Keyring 并瞬间切换后台 gRPC 会话上下文，真正做到不依赖外部管理工具的完全独立快捷切换。

## 💾 快速安装

> **注意**：本补丁采用**本地动态反编译重包技术**，只修补必要文件，不改动核心二进制，完美兼容后续官方的日常版本升级。
>
> 运行前请确保您的系统中已安装 Node.js (npm/npx) 环境（Antigravity 的用户一般都已经装了）。

### 方法 A：双击一键安装（推荐）
1. 下载本项目 ZIP 压缩包并解压。
2. 双击运行 **`安装汉化与优化补丁.bat`** 即可。
3. 安装程序会自动申请权限、关闭运行中的客户端、备份原版、执行解包覆盖和重新打包（全程约 3 秒）。
4. 重新打开客户端即可。

### 方法 B：PowerShell 手动安装
以管理员身份打开终端，在项目根目录下运行：
```powershell
Set-ExecutionPolicy Bypass -Scope Process -Force
.\install.ps1
```

## 🔄 一键卸载/还原 (v1.0.4 新增)

如果您想换回官方英文版，无需繁琐地去删除和改名文件：
1. 直接双击运行项目根目录下的 **`一键还原官方原版.bat`**。
2. 还原程序会自动申请权限、关闭运行中的客户端，并将备份的原版核心包一键还原。
3. 自动从开机启动项中注销并清理防覆盖自愈脚本，实现干净无残留卸载。

## 📁 项目结构

* `patch/`：优化与汉化补丁的源码
* `安装汉化与优化补丁.bat`：Windows 一键安装运行脚本
* `一键还原官方原版.bat`：Windows 一键还原官方原版运行脚本 [NEW]
* `install.ps1`：PowerShell 核心安装与 ASAR 打包器
* `restore.ps1`：PowerShell 核心还原与清理器 [NEW]
* `auto_heal.ps1`：动态防自动更新覆盖的开机自愈服务脚本
* `README.md`：本说明书
